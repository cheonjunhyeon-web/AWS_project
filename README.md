# IaC 기반의 멀티 리전 재해 복구 및 지능형 확장 인프라 구축
202-02-21일부터 시작  
24시간 중단 없는 서비스를 제공해야 하며, 특정 시간대에 트래픽이 몰리는 현상에 대한 대응하기 위한 환경 구성 
Multi-AZ 및 Multi-Region을 설계하여 가용성을 가지고 가며, Auto Scaling을 통해 트래픽 부하에 유연하게 대응하고 비용을 최적화한다.   
데이터는 S3수명 주기 정책으로 오래된 데이터 관리 및 멀티 리전 복제로 안정성을 확보한다.<br>
이 모든 과정을 Terraform 코드로 관리하여 휴먼 에러 방지 및 신속한 배포를 하는 것으로 프로젝트를 마무리 할 계획이다.
